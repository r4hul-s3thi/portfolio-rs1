# Rahul Sethi — Motion Portfolio

A modern, animated personal portfolio built with HTML, CSS & JavaScript.

🔗 **Live:** https://r4hul-s3thi.github.io/portfolio-rs1/

## Features
- Custom animated cursor
- Animated gradient blobs & particle canvas
- Typing effect
- Scroll-reveal animations
- 3D tilt project cards
- Animated skill cards
- Contact form
- Fully responsive
